### ejs

ejs是一种js的模板引擎

模板引擎：一种用于分离用户界面和业务数据的技术


实际上是前后断未分离时，为了分开 js变量 和 前端标签字符串 的一种方法

类似于一种模板字符串，

此时的模板可以是 html文件.toString() 

要插入的数据，作为第二个参数（对象的属性）传入

模板字符串

```
let y = 2023, m = 4, d = 15
let str = `今天是${y}年${m}月${d}日`
```

ejs

```
ejs.render(`<h2>
  今天是<%= y %>年<%= m %>月<%= d %>日
  </h2>`, { y: y, m: m, d})
```

注：ejs.render的第二个参数，若插入的js变量内外同名，可以简写 m: m ==> m
